模型厂商接入入口。

§线路协议（wire）

协议实现模块典型厂商
Chat Completions（OpenAI 兼容形）protocols::chat_completionsOpenAI、MiniMax、DeepSeek
Anthropic Messagesprotocols::anthropic_messagesClaude（Anthropic）

各厂商默认 URL、模型名、环境变量见 vendors；ProviderPreset::wire 决定用哪套协议。

新增厂商时：在 vendors 下增加模块，在 provider_presets 中注册，并设置正确的 WireProtocol。
